# Assessment Rubric
## AI-Powered Analyst Workflow Lab

---

## Participant Progress Rubric

| Criterion | 1 — Beginning | 2 — Developing | 3 — Proficient |
|---|---|---|---|
| **VS Code Navigation** | Cannot find files or open terminal independently | Opens files with repeated prompting, struggles with shortcuts | Navigates confidently — uses `Cmd+P`, opens terminal, splits panes |
| **Copilot Engagement** | Does not attempt Copilot or dismisses all suggestions | Uses Copilot with direct guidance, accepts suggestions without evaluating | Crafts targeted prompts, evaluates output critically, iterates |
| **Data Cleaning** | Script not run or `clean_sales.csv` not produced | Script runs with errors, no output file | Script runs cleanly — `clean_sales.csv` produced with correct transformations |
| **SQL / Python Analysis** | No queries attempted | Queries written but produce errors or incorrect results | At least 2 queries produce correct results with real numbers |
| **Insight Quality** | No specific insights documented | Vague observations ("North seems bigger") | 2+ specific findings with numbers ("North: $47K — 34% more than South") |
| **Report Quality** | Template not opened or blank | Partial completion — no executive summary | Full template: headline findings, executive summary, data quality notes |
| **Git Workflow** | No branch created, no commits | Branch created with instructor help or commit made without branch | `insight-[name]` branch visible — commit with descriptive message |

---

## Scoring Guide

| Score Range | Interpretation | Recommended Next Step |
|---|---|---|
| **18–21 points** | Ready for intermediate workshop | Workshop 2: Python for Business Analysts — pandas, visualization, automated reports |
| **12–17 points** | Solid foundation | Self-paced follow-up: complete take-home challenge, revisit Phase 4 |
| **Below 12** | Needs reinforcement | Revisit Phases 1–3 in a follow-up session before Workshop 2 |

## Pre/Post Confidence Survey

*Administer via Google Form or paper. Rate each item 1 (no confidence) to 5 (very confident).*

### Survey Questions (same before and after):

1. I can navigate a software project folder in VS Code
2. I can use GitHub Copilot to help me with a work task
3. I understand what a Git branch and commit are and why they matter
4. I can ask good questions to get useful insights from data
5. I understand the daily workflow my technical team uses

### Scoring Target

| Question | Pre-Lab Average | Post-Lab Average | Target Delta |
|---|---|---|---|
| VS Code navigation | ~1.5 | ~3.5 | +2.0 |
| GitHub Copilot | ~1.5 | ~3.5 | +2.0 |
| Git concepts | ~1.5 | ~3.0 | +1.5 |
| Data questioning | ~2.5 | ~4.0 | +1.5 |
| Team workflow | ~1.5 | ~3.5 | +2.0 |
| **Overall average** | | | **+1.8+** |

**Overall target:** Average delta of +1.5 or more across all 5 questions.

## Qualitative Post-Lab Questions

*(Open-ended — include on the same post-lab form)*

1. What was the most valuable thing you learned today? *(open text)*
2. What would you want to explore next? *(open text)*
3. Would you recommend this lab to a colleague? *(Yes / No / Maybe)*
4. What would make this lab better? *(open text)*

## 30-Day Follow-Up Survey

*Send 30 days after the session — keep it to 4 questions, under 2 minutes:*

1. Have you opened VS Code since the lab? *(Yes / No)*
2. Have you used GitHub Copilot since the lab? *(Yes / No)*
3. Has this lab changed how you work with your technical team? *(Yes / Somewhat / No)*
4. What one concept from the lab have you applied in your own work? *(open text)*
